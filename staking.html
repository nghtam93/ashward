Gift box

Gift box is an item rewarded to testnet event whitelist winners and anyone who participates in the ASC staking pool.

Each gift box contains 1 NFT and can be redeemed immediately after joining staking. This NFT will be available in game for players to play testnet.

For each minimum amount staked adhere specifically to every pool, user will receive a gift box.

Gift code

Gift code is an item rewarded to who is eligible for gift box and extend to users who use reference code to invite more users to join the game.

Gift code contains 1 item from the following pool: NFT, Free Awaken, Boxes of ASC, Pet (7 days), Decorative items.

These gift codes can be exchanged to real-valued items when the mainnet version is released.
